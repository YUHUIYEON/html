자바스크립트란?

    개발자가 만든 문서에 방문자가 방문하여 어떤 동작을 취하면 그 동작에 대하여 반응이 일어날 수 있도록 해주는 언어->액션이 실행될 수 있도록.
    1. 웹브라우저를 프로그램적으로 제어를 하기위한 언어
    2. 웹페이지를 동적으로 만들어줌
    3. 인터프리트 언어
        -> 작성된 순서대로 구문을 분석하고 에러가 발생하면 그 다음부터 구문 분석을 하지 않음
    4. 클라이언트 스크립트 언어
        ->서버에서 실행되는 것이 아니라 사용자 컴퓨터에서 실행

적용방식 3가지

    1. 스크립트 태그 사이에 기술 후 적용 ->내부스크립트
    2. 태그에 직접 적용 ->인라인 스크립트
    3. 외부파일 로드 적용 ->외부스크립트
