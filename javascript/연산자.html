    산술연산자
    +, -, *, / , %
    연산을 하기 위해서는 연산 대상 데이터가 반드시 2개 이상 있어야함.

    문자결합연산자
    피연산자가 문자형 데이터
    여러개의 문자를 하나의 문자형 데이터로 결합할 때 사용
    *피연자로 문자형 데이터가 한개라도 포함되어 있으면 다른 피연산자의 데이터는 자동으로 문자형 데이터로 형변환이 됨

    대입연산자(할당연산자)
    =, +=, -=, *=, /=, %=
    연산된 데이터를 변수에 저장할 때 사용
    결합방향은 오른쪽에서 왼쪽으로 결합하고 피연산자에 값을 할당

    증감연산자
    ++ -> 숫자형 데이터를 1씩 증가시키는 증가연산자
    -- -> 1씩 감소시키는 감소연산자 
    피연산자가 한개만 필요한 단항연산자

    변수의 어느위치에 오는가에 따라 결과값이 달라짐
    A = ++B; -> 전위 : 먼저 1을 더하고 변수의 값을 실행
    A = B++; -> 후위 : 변수의 값을 실행하고 1을 더함
    A = --B; -> 먼저 1을 빼고 변수의 값을 실행
    A = B--; -> 변수의 값을 실행하고 1을 빼기 실행

    비교연산자
    두데이터를 크다 작다 같다와 같이 비교할 때 사용
    연산된 결과값은 true, false 논리형 데이터로 반환

    == 동등연산자 -> 표현이 같으면 true를 반환
    === 일치연산자 -> 값과 자료형이 같아야 true 반환
                        데이터가 문자열이고 동등할 때는 알파벳순으로 비교해서 Boolean으로 반환
