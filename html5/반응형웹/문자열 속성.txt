문자열 속성 선택자
선택자[속성^=값]:속성안의 값이 col로 시작하는 단어를 선택
선택자[속성~=값]:속성안의 값이 특정 값을 온전한 단어로 포함하는 태그 선택
                 ex>ko-kr
선택자[속성|=값]:속성안의 값이 특정 값을 단어로 포함하는 태그 선택
                 ex>ko,kr
선택자[속성$=값]:속성안의 값이 특정값으로 끝나는 태그 선택
선택자[속성*=값]:속성안의 값이 특정값을 포함하는 태그 선택


그라디언트 넣기
CSS3는 두 가지 유형의 그라디언트를 정의합니다.
linear-gradient(아래 / 위 / 왼쪽 / 오른쪽 / 대각선으로 이동,컬러,컬러,컬러)
선형 그라디언트 (아래 / 위 / 왼쪽 / 오른쪽 / 대각선으로 이동,rgba(),rgba())
radial-gradient(중심에 정의 됨)
