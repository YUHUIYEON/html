DOM이란?

    DOM(Document Object Model)은 html문서의 요소를 제어하기 위해 브라우저에 지원이 됨
    DOM에서 D = document 문서 전체를 의미,
            O = object 객체 (element=>tag)
            M = model 문서를 해석하는 방식의 형태
    =>노드의 집합이라고 할 수 있음
    구성 : 요소노드(tag), 속성노드(attr), 텍스트(text=>내용)
  • 원산지 : 에디오피아
  • 지역 : 이르가체프 코체레
  • 농장 : 게뎁
