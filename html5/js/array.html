자바스크립트 배열이란?

  1. 연관된 데이터를 하나처럼 관리하고 싶을 때 사용
  2. 복수의 데이터를 제어할 수 있는 다양한 함수 제공
  3. 배열의 함수를 이용한 데이터의 편리한 조작 가능

배열의 선언 방법

    구문형식
    1. let array1=['element1','element2','element3',...]
    2. let array2=Array('element1','element2','element3',...)
    3. let array3=new Array('element1','element2','element3',...)

    배열의 호출 방법
    변수명[indexN] => array1[1]
