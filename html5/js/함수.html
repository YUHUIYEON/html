함수(function,method)

    1.일련의 로직을 다시 사용할 수 있도록 묶어둠(모듈화)
    2.입력(인자, ,argument), 출력(반환, return)으로 이루어짐
    3.내장함수, 사용자 함수로 구분함

    //기본형식
    function 함수명(인자1, 인자2, 인자3, ...){
        실행할 내용;
    }
    함수명 호출(인자1값, 인자2값, 인자3값, ...)

    //익명함수 : 함수명이 없는것을 말하고 함수선언문을 변수에 할당함
    let sum=function(a,b){
        return a+b;
    }
    document.write(sum(10,20));

    //일회성함수 : 한번만 실행할 수 있음
    (function(인자){
        실행할 내용;
    }(인자값));

    함수의 정의
    함수명은 함수를 호출할 때 사용하는 이름.(함수는 변수에 저장된다.)
    인자는 함수에 저장되는 변수이고, 콤마로 구분되고 없을수도 있고 무한히 많을 수도 있다.
    
    return  : 함수가 결과값으로 출력하는 값
            : 함수의 종료조건이 됨
