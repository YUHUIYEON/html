switch문

    지정한 변수에 저장된 데이터와 정확히 일치되는 경우가 있는지 검사하여 일치하는 데이터가 있을 경우 해당 실행문을 수행한 뒤 마지막의 break문으로 반복을 종료하여 빠져 나옴.

    break는 반복문을 즉시 중단을 시킬 때 사용.
    break 다르게 continue는 중단후에도 반복되는 조건이 남아있을 경우 반복문을 실행함.

    기본형식
    let 변수명=값;
    switch(변수){
        case 1 : 실행문;
        break;
        case 2 : 실행문;
        break;
        case 3 : 실행문;
        break;
        default:실행문;
    }
