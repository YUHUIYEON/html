제이쿼리란?

    제이쿼리는 자바스크립트의 생산성을 향상시켜주는 자바스크립트 라이브러리.
    - 라이브러리 -> 기존에 사용하는 코드들을 정의를 하여 프로그램의 효율성을 높이도록 만든 코드
    (잘쓰여지는 스크립트의 문법 및 형식을 코드화시켜서 정리해 놓은 파일을 라이브러리라 하고, 그 파일의 압축 내용을 실행시키는 것은 제이쿼리 메서드 작성으로 호출해서 실행하도록 한 것)

제이쿼리 특징

    1. 강력한 선택자 기능 제공
    2. 선택된 앨리먼트를 제어할 수 있는 함수 제공(메서드)
    3. 라이브러리화(반복적으로 호출해서 재사용이 가능)

제이쿼리 형식

    1. $('선택자').메서드();
        선택자 : id, class, tag(element), javascript object
    2. jQuery('선택자').메서드()
        플로그인에서 많이 사용
        누구누구를 선택해서 무엇을 할 것이냐? 는 뜻

제이쿼리 작성 순서

    1. 제이쿼리 라이블리 다운로드해서 경로잡기(CDN으로 경로 잡아주기)
        //맨위 상단에 있어야한다.

    2. 제이쿼리 랩퍼 선언하기
        -(document.ready(function(){실행문작성;}); -> 표준형식(요즘은 잘 쓰지않음)
        -$(function(){실행할 문장;}); -> 간략형식
        -jQuery(function(){실행할문장;}); -> 일회성
        -(function(){}(jQuery)); -> 일회성
        
        // 랩퍼를 생략하는 경우 : 플로그인을 문서내의 특정요소에 연결할 때
        // 랩퍼를 한번만 선언을 한다.

    3. 랩퍼안에 제이쿼리 형식으로 실행할 내용을 작성한다.

제이쿼리 부호($)의 역할

    제이쿼리에서 $()는 괄호안에 넣는 요소들을 선택하는 역할
    브라우저가 html tag를 모두 인식한 후 자동으로 실행되게 하는 역할 
    스크립트 코드가 꼬이는 오류를 방지하는 역할
